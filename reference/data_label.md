# Label factor variables using metadata

For each column of `x`, this function checks whether value labels exist
in `value_labels`. If so, integer values are replaced with these value
labels.

## Usage

``` r
data_label(
  x,
  variables = names(x),
  value_labels = read_yaml(paste0("value_labels_", substitute(x), ".yml"))
)
```

## Arguments

- x:

  A `data.frame`.

- variables:

  Column names of `x` to replace, Default: `names(x)`

- value_labels:

  A list with value labels, typically read from metadata generated by
  [`open_data`](https://cjvanlissa.github.io/worcs/reference/open_data.md)
  or
  [`closed_data`](https://cjvanlissa.github.io/worcs/reference/closed_data.md).
  Default: `read_yaml(paste0("value_labels_", substitute(x), ".yml"))`

## Value

A `data.frame`.

## Examples

``` r
if (FALSE) { # \dontrun{
if(interactive()){
 labs <- list(x = list(class = "factor", `1` = "a", `2` = "b"))

```
